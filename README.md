# Hack the North Frontend Challenge Starter

## Setup

### Install dependences:

```
npm install
```

### To Run:

```
npm run build
npm run start
```

### Run development mode:

```
npm run build-debug
```

Your app will be running at [http://localhost:5000/](http://localhost:5000/)
